# 👟 Footsy - Interfaz Web Responsive con Bootstrap

## Descripción

Footsy es un proyecto académico desarrollado como parte del programa **Tecnólogo en Análisis y Desarrollo de Software (SENA)**.

Este repositorio corresponde a la evidencia **GA5-AA1-EV07 - Elaboración de la Interfaz Gráfica para Aplicaciones Móviles**, donde se implementa la interfaz del sistema utilizando **HTML5, CSS3 y Bootstrap 5**, aplicando un diseño responsive adaptable a dispositivos móviles, tabletas y equipos de escritorio.

El objetivo es desarrollar una interfaz moderna, organizada y escalable que sirva como base para la implementación de la lógica de negocio en etapas posteriores del proyecto.

---

## Objetivo de la actividad

Diseñar e implementar la interfaz gráfica del aplicativo aplicando principios de usabilidad, accesibilidad y diseño responsive mediante el uso de Bootstrap, permitiendo una correcta adaptación a diferentes resoluciones y dispositivos.
